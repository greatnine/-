




# 如何应用C#9.0新特性?
===========================================================
C#9.0 添加一行

 < LangVersion > 9.0 </ LangVersion >

```
< PropertyGroup >
    < OutputType > Exe </ OutputType >
    < TargetFramework > net5.0 </ TargetFramework >

         < LangVersion > 9.0 </ LangVersion >

       < StartupObject > MySolution1.Program </ StartupObject >
         < UseWindowsForms > true </ UseWindowsForms >
     </ PropertyGroup >
< UseWindowsForms > true </ UseWindowsForms >

```
